Stash pHash Inspector

Reproduces and visualizes Stash's video perceptual hash, stage by stage, entirely on your machine.

1. Sample timestamps & frames

Each row is one video, scrolled horizontally in sample order. Rows scroll together, so the same position in each row is the same sample index — scroll any row to compare frame-for-frame. Click any frame to open a slider comparison against another loaded video at the source video's native resolution (a separate, unscaled screenshot at the same timestamp — only the collage stage below reflects the actual 160px-wide hash input).

2. Snapshot gridCollages (montage) — this is what actually gets hashed

The exact 5×5, 160 px-wide-tile pixels fed into the fingerprinthash. Placed side by side for comparison. Click a snapshot gridcollage to open a slider comparison against another loaded video's snapshot gridcollage, optionally overlaying which regions drove the fingerprintshashes apart.

3. Frequency patternsDCT coefficients (top-left 8×8, low frequencies)

Each cell represents a low-frequency pattern detected in the video. Warmer colors = stronger signal; 0/1 shows which side of average it falls on. Click a grid to see which patterns differ between two videos.

Warmer = larger magnitude. Each cell also shows whether it landed above (1) or below (0) the median. Click a grid to open a bit-level diff between two videos and see exactly which coefficients drove the hashes apart.
